A custom control which provides a multi-column list with varying row heights,
allowing multi-line text in the cells.  Cell text is wrapped as necessary and
may contain '\n' chars which will force line breaks.

See LICENSE.TXT for licensing information wrt using this code in your project.

To use, add MulilineList.h/.cpp to your project.  The other files in this
directory provide a demo project which shows the control in action and
illustrates how to use it in your project.
